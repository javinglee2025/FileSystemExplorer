# 更新日志
**广州智皓计算机技术有限公司**

## V0.3.9 (2026-04-20)

### 新增功能
- **文件浏览器右键菜单**
  - 支持对文件/目录执行打开、复制路径、十六进制查看等操作
  - 支持删除标记与恢复标记切换

- **十六进制查看器增强**
  - 新增保存按钮，支持将修改后的内容写回（需配合写入回调）
  - 新增多编码支持：ASCII、UTF-8、UTF-16、GBK、GB2312、GB18030、Big5、Shift_JIS、EUC-JP、ISO-8859 等
  - 搜索历史自动记录，支持快速回溯最近搜索关键词

### 改进
- **文件浏览器优化**
  - 大幅优化大目录加载性能
  - 修复 ICO 格式图标预览显示异常的问题
  - 修复十六进制预览标签页对齐问题

- **系统兼容性**
  - 修复 Windows 任务栏图标无法正常显示的问题
